## X11

- The classical method of displaying remote graphics.
- X11 forwarding lets you view graphical applications running on a remote server directly on your local machine.

**Pros:**

- Provides direct access to graphical applications from the remote server.
- Offers a familiar graphical interface for users.

**Cons:**

- Requires X server configuration on both local and remote machines.
- It can be slower and less efficient than other methods, especially over high-latency connections.
- May pose security risks if not properly configured and secured.
